# 🧮 Modern Calculator

A clean, responsive calculator built with **HTML, CSS, and JavaScript** — no frameworks, no libraries, just vanilla code. Designed to look and feel like a real physical calculator, with tactile button presses, an LCD-style display, and full keyboard support.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## ✨ Features

- **Responsive design** — works smoothly on mobile, tablet, and desktop screens
- **Realistic UI** — graphite body, warm ivory LCD-style display, and amber/teal accent keys for operators and equals
- **Tactile button feedback** — keys visually "press down" on click, like a real device
- **Full keyboard support** — use number keys, `+ - * /`, `Enter`, `Backspace`, and `Esc` directly
- **Error handling built in**
  - Divide-by-zero shows a clear message instead of `Infinity`
  - No duplicate decimal points or stacked leading zeros
  - Chained operations (e.g. `5 + 3 +`) calculate correctly
  - Floating-point rounding issues (e.g. `0.1 + 0.2`) are handled cleanly
- **Accessible** — visible keyboard focus states and ARIA labels on key controls
- **Zero dependencies** — just open `index.html` in a browser, no build step required

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in any modern browser

No installation, no dependencies, no build tools needed.

## 🛠️ Built With

- **HTML5** — semantic structure
- **CSS3** — custom properties (CSS variables), Grid layout, responsive media queries
- **Vanilla JavaScript** — DOM manipulation, event handling, no external libraries
